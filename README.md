A simple system to manage motorcycles and inventory, built with PHP and APIs.

---

## About

Track motorcycles, parts, and stock levels through a backend API and a simple frontend interface.

---

## Features

- Add, view, update, and delete motorcycles  
- Manage inventory via API endpoints  
- Simple, user-friendly frontend  
- Ready for extensions like authentication or reporting  

---

## Tech Stack

- **Backend:** PHP  
- **Frontend:** HTML, CSS, JavaScript  
- **Database:** MySQL
